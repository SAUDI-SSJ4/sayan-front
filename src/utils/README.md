# دليل استخدام التنبيهات الموحدة

تم إنشاء نظام موحد للتنبيهات في المنصة باستخدام مكتبة `react-hot-toast`. يهدف هذا النظام إلى توحيد شكل وأسلوب جميع التنبيهات في التطبيق لتحسين تجربة المستخدم وضمان اتساق واجهة المستخدم.

## الدوال المتاحة

تم توفير أربع دوال رئيسية للتنبيهات:

### 1. showErrorToast

تستخدم لعرض رسائل الخطأ بتصميم موحد.

```javascript
import { showErrorToast } from "../utils/toast";

// استخدام
showErrorToast("رسالة الخطأ هنا");
```

### 2. showSuccessToast

تستخدم لعرض رسائل النجاح بتصميم موحد.

```javascript
import { showSuccessToast } from "../utils/toast";

// استخدام
showSuccessToast("تمت العملية بنجاح");
```

### 3. showInfoToast

تستخدم لعرض رسائل المعلومات بتصميم موحد.

```javascript
import { showInfoToast } from "../utils/toast";

// استخدام
showInfoToast("معلومات مهمة للمستخدم");
```

### 4. showWarningToast

تستخدم لعرض رسائل التحذير بتصميم موحد.

```javascript
import { showWarningToast } from "../utils/toast";

// استخدام
showWarningToast("تحذير للمستخدم");
```

## مميزات النظام الموحد

- تصميم متناسق لجميع أنواع التنبيهات
- ألوان مختلفة لكل نوع من التنبيهات (أحمر للأخطاء، أخضر للنجاح، إلخ)
- موقع موحد للتنبيهات (أعلى المنتصف)
- مدة عرض موحدة (3 ثوانٍ)
- خط وحجم موحد للنص

## كيفية التحويل من النظام القديم

لتحويل التنبيهات من النظام القديم إلى النظام الجديد، استبدل:

```javascript
// قبل
import toast from "react-hot-toast";
toast.error("رسالة الخطأ");

// بعد
import { showErrorToast } from "../utils/toast";
showErrorToast("رسالة الخطأ");
```

وبالمثل لباقي أنواع التنبيهات (success, info, warning).

## ملاحظات هامة

- تأكد من تحديث المسار النسبي للاستيراد حسب موقع الملف الخاص بك
- لا تستخدم toast مباشرة من مكتبة react-hot-toast لضمان اتساق التصميم
- إذا كنت بحاجة إلى تخصيص إضافي، يرجى تحديث ملف `toast.js` بدلاً من إنشاء تنسيقات مخصصة في كل مكان
